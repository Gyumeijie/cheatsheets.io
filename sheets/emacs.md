## Shortcuts

<!-- {.-three-column} -->

### Movements

| Description               | Key             |
| ------------------------- | --------------- |
| Up/down                   | `C-n` _/_ `C-p` |
| left/right                | `C-f` _/_ `C-b` |
|                           |                 |
| Page up/page down         | `C-v` _/_ `M-v` |
|                           |                 |
| Beginning/end of line     | `C-a` _/_ `C-e` |
| Beginning/end of sentence | `M-a` _/_ `M-e` |

<!-- {.-shortcuts-right} -->

### Basic

| Description | Key         |
| ----------- | ----------- |
| Find file   | `C-x` `C-f` |
| Save file   | `C-x` `C-s` |
